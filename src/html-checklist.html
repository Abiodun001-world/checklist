Getting Started

HTML is the set of markup symbols or codes inserted into a file intended for display on the Internet.
  • Introduction
    HyperText Markup Language (HTML) is the language that powers the structure of every webpage you encounter. It's like the blueprint that defines how content is presented on the internet.
  • Types of HTML
    There are 4 different types of HTML file extensions and the extension denote a specific feature enabled in the HTML web page Types of HTML.
  • Doctype
    DoCTYPE is a declaration, but definitely not a tag .
  • HTML
    HTML and CSS are essential for web development and a good understanding of the key terms used is necessary.
  • XML
    XMLstands for Extensible Markup Language. It was developed to encode data and information in a format that could be easily interpreted by humans and computers alike.
  • Head Tag
    Head Tag in HTML contains all the header elements..
  • Body Tag
    Body Tag defines the body of the webpage.

HTML Attributes

HTML attributes provide additional information about HTML elements.
  • Id and Class name
    Id and Class, Class name can be used by multiple HTML elements, while an ID name must only be used by one HTML element within the page.
  • SVG
    SVG stands for Scalable Vector Graphics and it is one of the modern ways of displaying images in websites and applications just like displaying PNG, GIF, and JPG..
  • Images
    Images can be inserted into our HTML page using image tag. The image tag is an empty tag, which means that it does not have a closing tag. We can insert parameters of the image element using various attributes like src, alt, height and width..
  • Favicon
    A favicon is a small icon displayed left of the page title in a browser tab. It is loaded directly from the page requested, and is usually a logo representing the owner of the site.
  • Embed Tag
    The Embed Tag embed tag embeds interactive content or external application (plugin) in the specified part of HTML document.
  • Audio Tag
    Audio Tag element is used to embed sound or music element in our HTML page.
  • Video Tag
    Video Tag embeds a media player which supports video playback into the document. We can use video tag for audio content as well, but the audio tag may provide a more appropriate user experience..
  • Radio Button
    Radio Buttons are clickable circles that are a type of an input element in HTML.
  • MetaData
    The Metadata is the data (information) about data. In webpages, metadata comprises of key words and phrases that describe the content of the webpage.

Other HTML Elements

Other HTML elements below clearly describes its meaning to both the browser and the developer.
  • Heading
    Heading element implies all the font changes, paragraph breaks before and after, and any white space necessary to render the heading.
  • Button Tag
    Button Tag Buttons are like the ears of the websites we create.When they are clicked, they listen to our options or choices and subsequently required actions take place.
  • Span Tag
    Span Tag span is similar to all the other HTML elements, it starts with an opening tag span and ends with an closing tag span.
  • Lists
    Lists Lists are used commonly to create a set of data/information having common features using bullets/numbers.
  • Attribute Selectors
    Attribute Selectors in CSS allows you to select DOM elements based on the attributes they contain as well as the values of those attributes.
